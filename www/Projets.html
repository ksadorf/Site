Scrapathon de DataPublica

Python, expression régulière

Dans le cadre d'un concours organisé par DataPublica, le but était de scrapper puis extraire les délibérations du conseil de Paris en 4h. Il s'agit d'un projet réalisé à 3 et principalement en python. Les sources sont disponibles sur GitHub. Nous avons gagné le prix du public lors du concours.

Tower-Defense

C++, Qt

Un jeu de tower-Defense classique écrit en C++ en utilisant Qt. Il s'agit d'un projet d'école réalisé à deux dans le cours LO21.
Ce jeu contient toutes les mécaniques classiques d'un Tower-Defense: chemin d'ennemi, plusieurs types d'ennemi, plusieurs types de tourelle upgradable.
